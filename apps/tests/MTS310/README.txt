README for MTS310 test
Author/Contact: kvijayender@gmail.com, karnaty@iitk.ac.in

Description:

MTS310 is an application that tests various sensors in an MTS310
board.  The following is the list of sensors present in MTS310:

  Dual axis acceleroeter 
  Dual axis magnetometer 
  Light 
  Temperature  
  Acoustic 
  Sounder

Readings from each of these sensors are collected at regular intervals
and send to Basestation over the network.

Target hardware:
  iris (XM2100CA) + MTS310CB
